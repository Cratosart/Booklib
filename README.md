# Парсер книг с сайта tululu.org

Парсер онлайн библиотеки, скачивает текст книги в формате .txt
Так же выводит информацио об авторе книги, жанр и коментарии.

Книги скачиваются в папку `Books`
Изображения обложек в папку `images`
### Как установить

Для запуска нужен питон версии 3.8
Для установки всех необходимых модулей воспользуйтесь командой
```
pip install -r requirements.txt
```

### Аргументы
Парсер умеет принимать два параметра:

 `--start_page` - идентификатор книги с которой начать скачивать.
`--end_page`> - идентификатор до какой книги будет идти процесс скачивания.

### Пример запуска кода

```
python main.py 20 50
```
Данный пример скачивает книги с идентификаторами с 20 по 50, если они присутствуют в онлайн библиотеке
